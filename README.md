# clibf

## Install

### From PyPi

```sh
pip install clibf
```

### From GitHub

```sh
pip install git+https://github.com/donno2048/bf-interpreter
```

## Run

### Brainfuck file

```sh
clibf -f my_bf_script.b
```

### String

```sh
clibf -s "my_bf_string"
```

### Choose file with GUI

```sh
clibf -g
```
